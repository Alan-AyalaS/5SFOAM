¿Espumas 

5sfoam

Versátiles

Son una solución flexible y versátil que se utiliza para talleres y equipos de fábrica, transporte y embalaje o para la organización de oficinas, herramientas de alta calidad manteniendo el orden al máximo.

Diseñe las insercciones de espuma para cada herramienta para garantizar, el transporte, almacenamiento, etc.

Resistentes

Hechas 100% de Polietileno, ligeras como toda espuma pero resistentes como ninguna, las espumas 5sfoam son resistentes a golpes, al aceite, solventes comunes, corrosión, hongos, etc.

Además son 100% reciclables y libres de HCFC.
